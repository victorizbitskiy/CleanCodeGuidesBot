## Тестирование

### Название теста должно раскрывать его намерение

Когда тест не пройден, его имя является первым признаком того, что могло пойти не так.

**Плохо:**
```ts
describe('Calendar', () => {
  it('2/29/2020', () => {
    // ...
  });
  it('throws', () => {
    // ...
  });
});
```
**Хорошо:**
```ts
describe('Calendar', () => {
  it('should handle leap year', () => {
    // ...
  });
  it('should throw when format is invalid', () => {
    // ...
  });
});
```
